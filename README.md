# Spending Summary App

A full stack web application built with Angular and Node.js.

## Features
- Add and view expenses
- REST API integration
- Real-time UI updates

## Tech Stack
- Frontend: Angular
- Backend: Node.js, Express

## Run Locally

### Backend
cd backend
node server.js

### Frontend
cd ..
ng serve

## Highlights
- Built full stack architecture with Angular frontend and Express backend
- Implemented RESTful API communication using HttpClient
